# ArthSaarthi

**Personal Investment Portfolio Management System**

Welcome to the ArthSaarthi documentation!

## What is ArthSaarthi?

ArthSaarthi is a comprehensive, self-hosted personal investment portfolio management system that helps you track and analyze all your investments in one place.

## Key Features

- **Multi-Asset Support**: Track Stocks, Mutual Funds, Bonds, FDs, RDs, PPF, RSUs, and ESPPs
- **Real-Time Data**: Live price updates from major exchanges
- **Advanced Analytics**: XIRR, Sharpe Ratio, and portfolio performance metrics
- **Tax Lot Accounting**: Select specific lots when selling for tax optimization
- **Data Import**: Import transactions from broker statements (Zerodha, ICICI Direct)
- **Goal Tracking**: Set financial goals and link investments
- **Privacy Mode**: Hide sensitive values when sharing your screen
- **Backup & Restore**: Export and import your financial data
- **Multi-User**: Support for multiple users with admin controls

1. Getting Started

Welcome to ArthSaarthi! This guide will help you navigate through all the features of your personal investment tracker.

Logging In

Access the application through your browser. Enter your email and password to sign in.

Login page - Enter your credentials to access your portfolios
First Time Setup: If this is a fresh installation, the first user to register becomes the administrator.

2. Dashboard

Main dashboard showing portfolio value, P&L, allocation charts

Toggle privacy mode to hide sensitive financial information when sharing your screen.

Dashboard with privacy mode enabled - values are masked

Current market value of all your investments

Realized P&L

6. RSU & ESPP Tracking

Track Restricted Stock Units and Employee Stock Purchase Plan awards with proper tax lot accounting.

Add RSU/ESPP Award modal with Sell-to-Cover option
Sell to Cover: Record the automatic sale of shares used to cover taxes at vesting time.

Key Features

  • Track vest date and FMV at vesting
  • Record "Sell to Cover" transactions for taxes
  • Support for both US and Indian stock awards
  • Automatic cost basis tracking

7. Data Import

Bulk import transactions from broker statements (CSV format).

Data import page - upload CSV files from brokers

Supported Formats

  • Generic CSV
  • Zerodha contract notes
  • ICICI Direct statements
Ticker Mapping: The system will prompt you to map any unrecognized symbols to known assets.

10. Profile & Backup

Manage your account settings and backup your data.

Profile page with backup and restore options

Backup & Restore

Export your entire financial data to a JSON file for safekeeping. You can restore from this backup at any time.

Important: Restoring from a backup will replace all your current data. This action cannot be undone.

System maintenance - sync asset master data

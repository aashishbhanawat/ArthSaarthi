# Release Notes - v1.2.0

ArthSaarthi v1.2.0 brings significant improvements to portfolio visibility, tax optimization, and data import reliability.

## Key Highlights

### 1. Consolidated Holdings Table
The portfolio view now features a **groupable holdings table**. Assets are automatically categorized by type (Equity, Mutual Funds, Fixed Deposits, etc.) with collapsible sections, making it easier to manage complex portfolios.

### 2. Tax Lot Accounting & Specific Identification
Take full control of your capital gains. When selling assets, you can now:
- View all individual purchase lots (date, quantity, cost basis).
- Manually select specific lots to sell.
- Automatically calculate realized P&L based on your selection.

### 3. Enhanced Data Import Wizard
A complete overhaul of the import process:
- **Staging Preview:** Review and edit transactions before they hit your database.
- **Improved Resolvers:** smarter mapping for inconsistent ticker symbols from broker statements.
- **New Parsers:** support for advanced ICICI Direct and Zerodha statement formats.

### 4. Comprehensive Analytics & Benchmarking
- **Historical Snapshots:** Portfolio history now uses daily snapshots for lightning-fast rendering and accuracy.
- **Risk-Free Rate Overlay:** Compare your performance against a fixed risk-free benchmark on the chart.
- **XIRR Improvements:** Fixed edge cases in annualized return calculations for matured Fixed Deposits.

## Technical Improvements
- **Performance:** Optimized backend caching for large transaction histories.
- **UI/UX:** Native system tray integration for the Desktop application.
- **Security:** Enhanced backup/restore protocols with versioning support.

**Date:** 2026-01-15

We're excited to announce **ArthSaarthi v1.1.0**! This release focuses on enhanced analytics, improved desktop experience, and better documentation. Building on the solid foundation of v1.0.0, this update brings powerful new tools to help you understand your portfolio better.

## üöÄ How to Upgrade

### Desktop Users

Download the latest installer from [**GitHub Releases**](https://github.com/aashishbhanawat/ArthSaarthi/releases):

*   **Windows:** `ArthSaarthi-Windows-x64-Setup-v1.1.0.exe`
*   **macOS (Apple Silicon):** `ArthSaarthi-macOS-arm64-v1.1.0.dmg`
*   **macOS (Intel):** `ArthSaarthi-macOS-x64-v1.1.0.dmg`
*   **Linux:** `ArthSaarthi-Linux-x64-v1.1.0.AppImage` or `.deb`

**For macOS Users:** After copying the app to Applications, run:
```bash
xattr -cr /Applications/ArthSaarthi.app
```

### Self-Hosters (Docker)

Update your Docker images:
```bash
docker pull aashishbhanawat/arthsaarthi-backend:v1.1.0
docker pull aashishbhanawat/arthsaarthi-frontend:v1.1.0
```

Then restart your containers.

# ArthSaarthi v1.2.0 Release Notes

**Date:** March 31, 2026
**Version:** v1.2.0

## Highlights

This release focuses on **Stability, Security, and Advanced Analytics**. Key improvements include enhanced tracking for Fixed Income (FDs, RDs, Bonds, PPF), a more robust benchmarking engine with hybrid indices, and a comprehensive overhaul of the data import workflow.

### 📱 Android Application (Coming Soon)

An experimental Android mobile application is currently in development and is expected to be released soon! To bypass lengthy Play Store review processes initially, the Android APK will be published directly here on GitHub Releases. Stay tuned for updates!

### 🚀 New Features

- **Advanced Benchmarking (FR6.3):**
    - Compare your portfolio against **Hybrid Indices** (CRISIL 35/65 and 50/50 blends).
    - **Risk-Free Rate Overlay:** Visual dashed line on performance charts showing compound risk-free growth.
    - **Category Benchmarking:** Dedicated comparison for Equity (vs Nifty 50) and Debt (vs Bond Yields) with accurate segment-level XIRR.
- **Fixed Income Stabilization (v2):**
    - **Matured Asset Handling:** FDs and RDs now correctly transition out of active holdings while preserving historical interest data.
    - **Synthetic History:** Automatic injection of interest payout records into the transaction history for matured assets.
    - **Valuation Accuracy:** Resolved edge cases where non-market assets displayed $0 values on historical charts.
- **Data Import Enhancements:**
    - **Bank Statement PDF Import (FR7.2.1):** Import FD data directly from password-protected HDFC, ICICI, and SBI PDF statements.
    - **Symbol Alias Manager:** Admin UI to resolve broker-specific symbol naming conflicts (e.g., ICICI ShortNames) without manual code changes.
- **Performance & UX:**
    - **Daily Portfolio Snapshots:** Background pre-calculation of daily portfolio values, significantly improving dashboard load times.
- **Reporting & Compliance:**
    - **Demerger Cost Reduction:** Built-in support for calculating adjusted cost basis after corporate demergers (e.g., Reliance/Jio).
    - **Dividend Report Export:** Export your consolidated dividend reports directly to CSV for tax planning.
    - **Schedule 112A Enhancements:** Now includes expenditure and fees in capital gains calculations for improved accuracy.
- **Performance :**
    - **FIFO Optimization:** Refactored capital gains logic to handle large transaction histories up to 10x faster (O(N) complexity).
    - **Optimized Caching:** Significantly reduced dashboard load times via pre-calculated snapshots and efficient batch pre-fetching.

### 🐛 Key Bug Fixes

- Fixed Portfolio Realized P&L calculation distortions in historical views.
- Improved Data Import error messaging for clearer validation feedback.
- Fixed navigation and redirection issues when deleting linked portfolios.
- Resolved dark mode visibility issues for charts and input fields.

## Installation & Upgrade

- **Desktop users:** Download the latest installer for your platform from the [Releases Page](https://github.com/aashishbhanawat/ArthSaarthi/releases).
- **Server users:** Update your `docker-compose.yml` to use the `v1.2.0` tag or run `docker compose pull`.
